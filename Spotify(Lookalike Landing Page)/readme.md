# Spotify Lookalike Landing Page

A modern Spotify-inspired landing page built using HTML and CSS.

## Overview

This project is a frontend clone inspired by Spotify’s official landing page design.  
It focuses on responsive layouts, modern UI styling, clean structure, and smooth user experience.

## Features

- Responsive design
- Modern dark-themed UI
- Hero landing section
- Navigation bar
- Styled buttons and sections
- Smooth scrolling
- Mobile-friendly layout
- Clean HTML & CSS structure

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was created to practice:

- Frontend web development
- Responsive design techniques
- UI/UX styling
- Git & GitHub workflows
- GitHub Pages deployment
