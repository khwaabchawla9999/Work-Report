# Linux + Git Basics

A practical learning repository focused on Linux fundamentals, terminal usage, file permissions, process management, package management, Git workflows, and GitHub version control.

## Overview

This section contains notes, commands, practice exercises, and cheat sheets related to Linux administration and Git/GitHub operations. It serves as a hands-on reference for understanding real-world development and system management workflows.

## Topics Covered

- Linux terminal commands
- File & directory management
- User and permission handling
- Process management
- Package managers
- Networking basics
- Git fundamentals
- Git branching & commits
- GitHub repositories
- Push, pull & merge workflows
